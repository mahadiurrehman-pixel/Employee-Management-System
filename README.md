# 👨‍💼 Employee Management System

A professional CLI-based Employee Management System built with **Python**, **SQLite**, and **Object-Oriented Programming (OOP)**. This project allows users to manage employee records efficiently through a menu-driven interface.

---

## 🚀 Features

- 🔐 User Login & Registration
- ➕ Add Employee
- 📋 View All Employees
- 🔍 Search Employee
- ✏️ Update Employee Details
- 💰 Update Employee Salary
- ❌ Delete Employee
- 📊 Employee Statistics
- 🏆 Highest Paid Employee
- 📁 Export Employee Records to CSV
- 📧 Email Validation
- 📱 Phone Number Validation
- 💾 SQLite Database Storage

---

## 🛠️ Technologies Used

- Python 3
- SQLite3
- CSV Module
- Regular Expressions (Regex)
- Object-Oriented Programming (OOP)
